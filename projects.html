May Velas

We develop a web application with the basic characteristics of an E-commerce. This application has a great advantage and it is the versatility to scale to any sales project, in this case, it has been scaled to a scented candle venture. The application allows you to view a list of products, filter them by type, select the number of products, then, when you go to the shopping cart view you can see the total price of the purchase, to finish the purchase, ask for the shipping data and send a whatsapp message to the company number that is predetermined in the application.

https://4nd1n05.github.io/mayVelas/
